Dieses Plugin erweitert Elementor Pro um maßgeschneiderte Widgets, entwickelt von der VIERLESS GmbH.

== Description ==

Alfreds Toolbox ist eine exklusive Sammlung maßgeschneiderter Elementor-Widgets, entwickelt von [VIERLESS GmbH](https://vierless.de/) für unsere Kunden. Diese Erweiterung verbessert Ihre Elementor Pro Installation mit spezialisierten Widgets, die für maximale Benutzerfreundlichkeit und Leistung optimiert sind.

= Hauptfunktionen =

* Spotify Podcast Widget - Integrieren Sie Ihre Podcast-Episoden nahtlos in Ihre Website

= Vorteile =

* Nahtlose Integration in Elementor Pro
* Regelmäßige Updates und Support
* Optimierte Performance
* Responsive Design für alle Geräte
* Kundenspezifische Anpassungen möglich

= Systemanforderungen =

* WordPress 6.6.2 oder höher
* PHP 8.0 oder höher
* Elementor Pro (aktuelle Version)

== Installation ==

= So funktioniert's =
* Laden Sie Alfreds Toolbox auf Ihre WordPress-Website hoch
* Installieren Sie Elementor Page Builder
* Nutzen Sie die erweiterten Widgets nach Belieben
